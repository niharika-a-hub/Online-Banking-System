
# Online Banking System 🏦
A console-based Java application designed to handle basic banking operations.

## ✨ Features
- **Account Management:** Create and view account details.
- **Transactions:** Secure deposit and withdrawal logic.
- **Data Validation:** Prevents negative deposits or over-drawing.

## 🛠️ Tech Stack
- **Language:** Java
- **Concepts:** Object-Oriented Programming (OOP), Encapsulation, Exception Handling.
